# THACO

aka **[T]** o **[H]** it **[A]** rmor **[C]** lass **[Z]** ero

An [Owlbear.Rodeo](https://www.owlbear.rodeo/) extension for calculating your THACO.

## Development

The Node.js project is located in the `THACO/` subdirectory. You must navigate into this directory before running any of the following Node commands:

```bash
cd THACO
```

### Basic Commands

- **Install dependencies:**
  ```bash
  npm install
  ```

- **Start development server:**
  ```bash
  npm run dev
  ```

- **Build project:**
  ```bash
  npm run build
  ```

- **Run unit tests:**
  ```bash
  npm test
  ```

- **Run UI tests (requires Playwright setup):**
  ```bash
  npm run test:ui
  ```
